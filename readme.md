# Elegant Slider v0.5

This JS/CSS module will generate a slider based on a given div parent node.  
Each first child of the parent will be displayed as a slide.  
It's compatible with touch events.  
I tried my best to make it simple, robust and light-weighted.

## Steps
1. import `slider.js` and `slider.css` files into your html page.
2. execute function `createSlider(node, options)`

- `node` is the HTML node that will be displayed as slider. Its direct children will be displayed as slides.
- `options` is an object with the following elements. If not given to the function, it will use default values.

## Options

```
{  
	width: slider width (string. Every CSS units are valid, default = 100%),
	height: slider height in % of width (number. Default = 56.25. if 0, height will fit content),
	animation: translate or fade (string. Defaut = "translate"),
	animationTiming: animation duration in ms (number. Default = 500),
	pauseTiming: slide display duration in ms (number. Defaut = 0, i.e manual switch between slides),
	initialPosition: slide to begin with (number. Default = 0, i.e first slide),
	infiniteCycle: infinite navigation (boolean. Default = false),
	showArrows: show navigation arrows (boolean. Default = true),
	showDots: show navgation dots (boolean. Default = true),
}
```
## Notes
- CSS is minimal. Feel free to tune it to your needs. Some parameters are generated by JS though.
- Each direct child of the parent will be displayed as a block.
- With fading animation, due to absolute position of slides, **height of slider must be specified** (default value is okay, or specified in your own CSS).
